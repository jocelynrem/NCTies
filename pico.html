DevTools Steps

Mac: Cmd + Option + I
PC: Ctrl + Shift + I

ink_selection Select element: click this, then click a part of the page to find its name/class.

devices Toggle device toolbar: preview how the page looks on phone/tablet widths.

Action 1: Find the element name for AI

  1. Open DevTools (shortcut above or Right-click → Inspect)
  2. Select element, then click the part you want to change
  3. Find its name/class (example: article or .callout)
  4. Ask AI: “Change only this part: .callout”
  5. Paste AI's update into your file → refresh

Action 2: Check different screen sizes

  1. Click Toggle device toolbar
  2. Switch between phone/tablet widths
  3. Ask AI to adjust spacing or layout if anything looks off

Design System: Pico

Pico is a minimal CSS framework that emphasizes typography + spacing. It’s great when your page is mostly text and links.

  • Best for: unit hubs, resource collections, parent info pages
  • AI-friendly: less class noise → easier for beginners to edit
  • Teacher takeaway: “clean and readable” with almost no setup
