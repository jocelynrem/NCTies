Page Design Basics (No coding background needed)

Use this as a cheat sheet when you ask AI to build or edit a page. Learn a few key terms and visual patterns so your prompts are clear.

How to ask AI CSS frameworks UI elements DevTools Accessibility Padding vs margin

1) Prompt AI in plain language

Clear intent beats technical jargon. You do not need \"developer words\" to get good results.

A strong beginner prompt has 3 parts: what to change, where to change it, and what to keep the same.

Example prompt

In the resources section card, add more padding inside each card and more margin between sections.
Keep the current colors and typography.
Only edit the card and spacing styles.
Tip: Need an exact element name? Use the DevTools steps in section 6, then include that name in your prompt for more precise edits.

2) CSS frameworks (the simple version)

A CSS framework is a shortcut to a better-looking page. Instead of styling everything from scratch, you use pre-built layout and UI styles (buttons, cards, spacing) so your page looks polished fast.

Framework

Bootstrap

Best when you want a polished, \"finished app\" look quickly.

Framework

Bulma

Best when you want clean layout with easy-to-read class labels.

Framework

Pico

Best when you want simple, minimal code with great default text styling.

Why this helps with AI coding: AI already understands these CSS frameworks. You can ask it to \"build this with Bootstrap,\" or \"swap this page to Bulma/Pico,\" and it can restyle the same content much faster and more consistently.

4) DevTools: 2 actions you need

DevTools can do a lot, but we'll only do two tasks: find the exact element, and preview different screen sizes.

Open DevTools

Mac: Cmd + Option + I

Windows/PC: Ctrl + Shift + I

Action 1

Select element

Click the pointer icon, then click the page part you want to edit. Use that name in your AI prompt.

Action 2

Toggle device toolbar

Check phone/tablet views to spot spacing and readability issues on smaller screens.

5) Quick accessibility checklist

Accessibility means making your page easier for more people to read, use, and navigate.

Treat this as a quick confidence check, not a full audit. If something looks off, ask AI to fix that one item.

check Use clear heading order: h1, then h2, then h3.

  • Improves readability for everyone.
  • Helps AI and tools target edits accurately.

Prompt hint: "Make sure the heading order is screen reader accessible."

check Keep body text easy to read (about 16px).

  • Small text is harder to read on phones.
  • Larger text reduces strain for low-vision users.

Prompt hint: "Increase body text to around 16px and keep line spacing readable."

check Make sure text stands out from the background.

  • Low contrast text looks faded.
  • Better contrast improves quick readability.

Prompt hint: "Improve text/background contrast in this section so content is easier to read."

check Make links and buttons easy to spot and understand.

  • Users can quickly tell what is clickable.
  • Clear labels reduce confusion.

Prompt hint: "Make all links and buttons look clearly clickable and use clear labels."

check Use simple motion: avoid busy animations and respect prefers-reduced-motion.

  • Heavy motion can distract or overwhelm users.
  • Reduced-motion settings improve comfort.

Prompt hint: "Reduce animation intensity and add prefers-reduced-motion support."

check Press Tab and check that focus is visible.

  • Keyboard users need to track position.
  • Visible focus prevents navigation mistakes.

Prompt hint: "Improve focus styles so keyboard navigation is always easy to see."

Bonus tip

Padding vs Margin (easy rule)

Use this simple rule: padding is space inside a box, margin is space outside a box. Using the right word in your AI prompt helps AI change the correct spacing faster.

Margin (outside space)
Padding (inside space)
Content (text, button, image)

Prompt wording you can copy

  • "Add more padding inside this card."
  • "Add more margin-top between these sections."

Quick choice: if content feels cramped, use padding. If sections feel too close together, use margin.
