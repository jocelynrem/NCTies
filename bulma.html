DevTools Steps

Mac: Cmd + Option + I
PC: Ctrl + Shift + I

ink_selection Select element: click this, then click a part of the page to find its name/class.

devices Toggle device toolbar: preview how the page looks on phone/tablet widths.

Action 1: Find the element name for AI

  1. Open DevTools (shortcut above or Right-click → Inspect)
  2. Click Select element, then click what you want to change
  3. Find the class or element name (example: .button)
  4. Ask AI: “Change only the .button section”
  5. Paste AI's update back into your file → refresh

Action 2: Check different screen sizes

  1. Click Toggle device toolbar
  2. Switch between phone/tablet widths
  3. Ask AI to adjust spacing or layout if anything looks off

Design System: Bulma

Bulma is a CSS framework with human-readable class names that feel approachable. It’s great when teachers want a clean site without lots of “extra UI.”

  • Best for: class hubs, resource pages, simple professional layouts
  • AI-friendly: predictable components and straightforward markup
  • Teacher takeaway: easier to read and tweak than many frameworks
