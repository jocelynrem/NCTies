Resource Challenges

Use these quick challenges to choose a color theme, icon style, and UI pattern before you build your Frankenstein page.

3 quick challenges Beginner-friendly Create Frankenstein

Challenge 1: Pick a color theme

Open Coolors and choose one palette you like.

  • Choose 4-5 colors
  • Save the hex codes
  • Mark one background color and one accent color

Challenge 2: Choose an icon library

Browse Font Awesome and pick a simple icon style for your page.

  • Pick 3-5 icons that fit your topic
  • Use one icon for buttons or links
  • Keep icon style consistent across the page

Challenge 3: Choose one UI pattern

Look through UIverse and pick one card/component idea you want to adapt.

  • Pick only one pattern to start
  • Choose a beginner-friendly option
  • Match it to your chosen colors and icons

Copy/Paste Prompt (customize your favorite page)

Pick your favorite framework page first (Pico, Bootstrap, or Bulma), then copy this prompt. 

Use my [Pico OR Bootstrap OR Bulma] page as the starting point for customization.
Keep the same page topic and the same core sections/content.

Apply these updates:
1) Use this color palette: [paste 4-5 hex colors from Coolors]
2) Use icons from Font Awesome for at least:
   - one section heading
   - one button or call-to-action
   - one list item or card
3) Add one UI pattern inspired by UIverse (for example: a feature card style, callout panel, or button style)

Constraints:
- Keep it beginner-friendly and readable
- Do not add complex animations
- Keep accessibility basics (clear heading order, keyboard focus visible, good text contrast)

Output a single updated HTML file called frankenstein.html
