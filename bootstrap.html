Workshop Prompt (copy/paste)

Use this to generate your first version before swapping frameworks. 

Create a single-page workshop landing page for “AI Coding with DevTools + Design Systems” using the Bootstrap 5 framework.
Use the Bootstrap 5.3 CDN and Bootstrap components/utilities for layout and styling (container, row/col grid, navbar, cards, buttons, badges, spacing classes).
Include: a hero header, three feature cards (Prompting, Styling, Debugging), a resources section with links, and a “Try This Challenge” callout.
Use semantic HTML and basic accessibility (headings, skip link, keyboard focus styles).
Output a single bootstrap.html file.

DevTools Steps

Mac: Cmd + Option + I
PC: Ctrl + Shift + I

ink_selection Select element: click this, then click a part of the page to find its name/class.

devices Toggle device toolbar: preview how the page looks on phone/tablet widths.

Action 1: Find the element name for AI

  1. Open DevTools (shortcut above or Right-click → Inspect)
  2. Click Select element, then click what you want to change
  3. Find the class or element name (example: .card-body)
  4. Tell AI: “Change only the .card-body section”
  5. Paste AI’s update into your file → refresh

Action 2: Check different screen sizes

  1. Click Toggle device toolbar
  2. Switch between phone/tablet widths
  3. Ask AI to adjust spacing or layout if anything looks off

Design System: Bootstrap

Bootstrap shines when you want a “finished UI” quickly: navbars, cards, buttons, spacing utilities, and responsive layout.

  • Best for: quick professional layouts, dashboards, school “hub” pages
  • AI-friendly: predictable classes + tons of examples in docs
  • Teacher takeaway: less design guessing — more consistent structure
