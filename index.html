Workshop Goal

Use AI to generate code, then use DevTools only to find element names so AI can make targeted updates.

45 minutes Beginner-friendly Copy/paste ready

Flow

  • Generate a simple page
  • Swap in a design system
  • Use DevTools to identify the element/class name
  • Publish in Google Sites/Canva

Tools

  • Colors: Coolors, Adobe Color
  • Icons: Font Awesome, Icons8, SVG Repo
  • Patterns: UI Design Patterns article , UXPin UI Elements article
  • Components: UIverse
  • Design systems: Bootstrap docs, Bulma docs, Pico CSS docs
  • Typography: Google Fonts

Workshop Prompt (copy/paste)

Use this to generate the first version. Then we’ll restyle it with a design system. 

Create a single-page workshop landing page for “AI Coding with DevTools + Design Systems.”
Include: a hero header, three feature cards (Prompting, Styling, Debugging), a resources section with links,
and a “Try This Challenge” callout. Use semantic HTML and basic accessibility (headings, skip link, focus).
Output a single index.html file.

Design Systems (what you’ll see today)

  • Bootstrap: “App-like” UI with strong components and layout utilities.
  • Bulma: Readable class names and a clean, content-first layout.
  • Pico: Minimal classes with great default typography and spacing.

This baseline page is intentionally plain so the differences pop on the other versions.
