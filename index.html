Workshop Goal

Build one simple page with AI, then improve it in small steps. Learn how CSS frameworks change the look without changing your core content.

45 minutes Beginner-friendly Keep it simple

Workshop Flow

  • Create a basic starter page with AI
  • Try the same page in Pico
  • Try the same page in Bootstrap
  • Try the same page in Bulma
  • Mix ideas and customize in Frankenstein

Tools

  • AI chat tool (ChatGPT, Copilot, Claude, etc.)
  • Browser + DevTools Inspect tool
  • Color tools: Coolors, Adobe Color
  • Icons: Font Awesome, SVG Repo
  • Patterns: UI Design Patterns article , UXPin UI Elements article
  • Components: UIverse
  • CSS frameworks: Bootstrap docs, Bulma docs, Pico CSS docs
  • Typography: Google Fonts

Starter Prompt (copy/paste)

Keep your first page quick and simple. You will improve it in later workshop steps.

Copy the prompt below exactly, then run it in your AI tool. 

Create a very simple one-page website for a school-friendly topic (such as a class pet fan page, reading challenge tracker, hobby club, or lunch menu highlights). Include a page title/header, a short intro paragraph, three content sections, one list of links/resources, and one button or call-to-action. Use plain semantic HTML with basic accessibility (headings in order and a skip link). Keep the design basic and clean with no advanced animations or complex layout.
Output a single index.html file.

Choose your own page idea (optional)

You can use your own idea instead of the sample topic.

  • Keep it to one page
  • Use only a few sections
  • Avoid advanced features for now
  • Focus on clear content and structure first

What happens next

  • Pico: minimal, clean defaults with very little class setup.
  • Bootstrap: strong components and utility classes for polished layouts.
  • Bulma: readable class names and a content-first structure.
  • Frankenstein: mix styles and personalize your final version.

You are not trying to perfect this page yet. Keep the starter version basic so the framework differences are easy to notice.
