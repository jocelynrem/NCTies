Workshop Prompt (copy/paste)

Use this to generate the first version. Then we’ll restyle it with a design system. 

Create a single-page workshop landing page for “AI Coding with DevTools + Design Systems.”
Include: a hero header, three feature cards (Prompting, Styling, Debugging), a resources section with links,
and a “Try This Challenge” callout. Use semantic HTML and basic accessibility (headings, skip link, focus).
Output a single baseline.html file.

DevTools (quick steps)

Mac: Cmd + Option + I
PC: Ctrl + Shift + I

ink_selection Select element: click this, then click a part of the page to find its name/class.

devices Toggle device toolbar: preview how the page looks on phone/tablet widths.

Action 1: Find the element name for AI

  1. Open DevTools (shortcut above or Right-click → Inspect)
  2. Use Select element, then click the part you want to change
  3. Find the class/element name
  4. Tell AI: “Edit only this class/element”
  5. Replace in your file → refresh

Action 2: Check different screen sizes

  1. Click Toggle device toolbar
  2. Switch between phone/tablet widths
  3. Ask AI to adjust spacing or layout if anything looks off

Design Systems (what you’ll see today)

  • Bootstrap: “App-like” UI with strong components and layout utilities.
  • Bulma: Readable class names and a clean, content-first layout.
  • Pico: Minimal classes with great default typography and spacing.
